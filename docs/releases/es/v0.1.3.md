# Kumoriya v0.1.3 — Notas de Versión Alpha

**Fecha:** 2026-04-01

## Novedades

### Explorar y descubrir
- **Explorar anime con más control** — filtra por género, formato y orden, incluso eligiendo varios géneros a la vez.
- **Buscador por tags de AniList** — navega los tags organizados por categoría para encontrar anime que encaje mejor con lo que buscas.
- **Descubrimiento estacional en una sola vista** — junta anime de temporada actual, próximos y recomendados en una sola consulta.

### Soporte y feedback
- **Botón de reporte de bug** — envía comentarios y problemas directamente al equipo con más contexto.
- **Monitoreo de errores** — ayuda a detectar problemas más rápido para poder corregirlos antes.

### Mejoras del reproductor
- **Seek acumulado** — los saltos rápidos ahora se acumulan para avanzar o retroceder con más precisión.
- **Auto-next más seguro** — evita que el siguiente episodio se active otra vez por error.
- **Cierre del reproductor más estable** — mejora el cierre de la reproducción para evitar fallos raros.

### Matching de anime
- **Mejor coincidencia de títulos** — algunos nombres con guiones ahora se reconocen correctamente.
- **Más opciones de reproducción** — se añadieron espejos extra para tener más posibilidades de ver el contenido.

### AniList y almacenamiento
- **Más consultas útiles** — la app ahora puede pedir más datos de géneros, tags y listas de anime.
- **Historial de reproducción** — mejor gestión para ver, borrar o limpiar tu historial.
- **Menos llamadas de red** — varias consultas ahora se agrupan para ir más rápido.
