# Kumoriya v0.1.3 — Alpha Release Notes

**Date:** 2026-04-01

## What's New

### Browse & discovery
- **Browse Anime page** — filter by genre, format, and sort type, with support for multiple genres at once.
- **Tag-guided Anime Finder** — explore AniList tags by category to find anime that matches what you want.
- **Consolidated seasonal discovery** — current-season, upcoming, and recommended anime in one place.

### Support & feedback
- **Bug report button** — send feedback and issues directly to the team with more context.
- **Error monitoring** — helps catch problems faster so they can be fixed sooner.

### Player improvements
- **Seek accumulation** — quick taps now build up, making it easier to jump around more precisely.
- **Safer auto-next** — prevents the next episode from firing again by mistake.
- **More stable teardown** — improves shutdown behavior and avoids rare player crashes.

### Anime matching
- **Better title matching** — some hyphenated titles are now recognized correctly.
- **More playback options** — extra mirror hosts were added to improve availability.

### AniList & storage
- **More useful queries** — the app can now request more genre, tag, and anime list data.
- **Watch history management** — better support for viewing, deleting, or clearing your history.
- **Fewer network calls** — several requests are now grouped together for faster loading.

## Platforms
- Android (APK)
- Windows (Installer)
