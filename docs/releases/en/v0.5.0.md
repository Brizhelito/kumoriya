# Release v0.5.0

Date: 2026-06-14

## Summary
This release introduces the Miruro anime source with four resolvers, a major Watch Party redesign with immersive re-entry and player HUD, and comprehensive system documentation.

## Added
- **Miruro source plugin** — new anime source with four resolvers (AniDB, Kwik, VibePlayer, VidTube) for expanded content availability.
- **Watch Party immersive re-entry** — exit confirmation dialog, active party banner in the main UI, and player HUD integration for seamless party management.
- **System documentation** — comprehensive guides covering architecture, backend, CI/CD, data flow, edge infrastructure, frontend, plugin system, and testing.
- **MIT LICENSE** file added to the repository.

## Changed
- **Watch Party navigation** — migrated from push-based lobby to pop-based navigation with player HUD, making party flow feel more natural.
- **Player page redesigned** — improved controls layout and playback flow for a cleaner experience.
- **Party session architecture** — refactored session guard, realtime state, and providers for better reliability and state consistency.
- **Party lobby page retired** — replaced by the new navigation flow; all party actions now accessible from context.
- **Localizations** — updated English and Spanish strings for all new features.

## Fixed
- **Watch Party lobby navigation** — pushReplacement correctly restored after visiting anime detail page.
- **PartyRoomDO** — improved connection state restoration and reconnection handling for more robust party sessions.

- None.
