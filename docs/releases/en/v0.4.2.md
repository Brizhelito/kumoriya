# Release v0.4.2

Date: 2026-06-06

## Summary
This patch release focuses on player stability and catalog reliability, specifically addressing issues with split-season episode alignment and device orientation behavior.

## Fixed
- **Player Lifecycle Robustness**: Video playback now only pauses when the app is fully backgrounded, preventing intrusive pauses during orientation changes or system tray interactions.
- **Orientation Lock Control**: Centralized orientation management to ensure the lock behaves predictably and resumes auto-rotation correctly when disabled.
- **Split-Season Episode Synthesis**: Fixed a truncation bug where extra episodes from sources combining multiple seasons (S1+S2) were discarded if the corresponding AniList entry was out of date.
- **Catalog Cache Reliability**: Improved refresh logic for grouped-season title matches to ensure newly released episodes are promptly visible.
- **Episode Title Normalization**: Enhanced title parsing to treat season-specific source labels as generic episode titles for better visual consistency.

## Added
- **Diagnostic Logging**: Added internal instrumentation for source availability tracking and cache performance monitoring.
