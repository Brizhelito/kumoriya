# KUMORIYA DESIGN SYSTEM v2

## Overview
Premium anime streaming UI/UX system designed for mobile-first, scalable to desktop.

---

## 1. Core Principles
- Fast access to playback (≤2 taps)
- Minimal friction
- No placeholders
- Real data only
- Mobile-first, desktop-native adaptation
- Clean, premium, dark UI

---

Success: #00E676

## 12. Product Fidelity Rules
- No placeholders
- No fake data
- No broken flows
- No technical labels
- Only actionable UI

---

## 13. UX Rules
- Max 2–3 steps to playback
- Continue Watching always visible
- Episode access immediate
- Minimal decision friction

---

## 14. Future Ready
- Manga/Manhwa support
- Plugin-based sources
- Expandable player system

## Final Goal
Deliver a **premium streaming experience** optimized for anime consumption, fast interaction, and clean design.
