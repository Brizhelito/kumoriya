# Kumoriya - TODOs diferidos

Índice de trabajos identificados pero pospuestos conscientemente. Cada archivo
describe **qué hay hecho**, **qué falta** y **consideraciones** para retomar el
trabajo sin tener que redescubrir el contexto.

Reglas:

- Un archivo por tema, en `kebab-case.md`.
- Mantener el estado actualizado cuando se avance o descarte algo.
- Al cerrar un TODO, mover el archivo a `docs/todo/done/` o borrarlo y dejar
  nota en el changelog / dev-diary.
