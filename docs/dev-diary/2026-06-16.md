## 16:00 — kumoriya:// custom URI protocol in Inno Setup installer

**Category:** feature
**Affected files:**
- `apps/kumoriya_app/windows/kumoriya_installer.iss`

**Summary:**
Added `[Registry]` section to the Inno Setup script that registers the `kumoriya://` custom URI protocol on install. Uses HKCU keys (`SOFTWARE\Classes\kumoriya`) so no admin rights are required. The `URL Protocol` value tells Windows Shell this is a pluggable protocol handler, and `shell\open\command` points to `kumoriya_app.exe "%1"` so the OS passes the full URI as the first CLI arg. All keys carry `uninsdeletekey` so they are cleaned up automatically on uninstall.
