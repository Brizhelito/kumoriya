# Codex Prompt - Bootstrap Kumoriya Repo

Goal:
Create the initial Kumoriya monorepo with Flutter app and core packages.

Requirements:
- Android-first, Windows supported
- plugin-first architecture
- AniList as canonical metadata source
- Riverpod
- feature-first modular monolith
- tests and analysis wired from day 1

Guardrails:
- do not copy legacy code blindly
- do not implement playback or resolvers in the bootstrap
- create commits by phase
